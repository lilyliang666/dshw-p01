### 数据来源
- 股票行情：baostock，后复权，日度
- 市场指数：沪深300（000300）作为市场基准，中证500（000905）作为补充指数
  - 中证500选择理由：代表中小盘股票表现，与沪深300（大盘）形成互补，能更全面地观察A股市场不同市值风格的表现。
- 宏观指标：
  - CPI同比增速，来源：世界银行 API (World Bank, FP.CPI.TOTL.ZG - Inflation, consumer prices)
  - 人民币/美元汇率，来源：世界银行 API (World Bank, PA.NUS.FCRF - Official exchange rate)
  - 选择汇率的理由：汇率影响跨国企业盈利和国际资本流动，与股市表现密切相关
  - 注：世界银行只有年度数据，已分配到各月
- 财务数据：baostock,ROE和净利润率


### 存储方式
- 基础：CSV（方式 A）
  - CSV 优点：通用性强，几乎所有工具都支持；可读性好，可用文本编辑器直接查看；简单易用。
  - CSV 局限性：不支持类型契约，读取时需重新推断类型；压缩效率低，文件体积大；不支持列式读取，无法只加载需要的列；写入速度较慢。
- 进阶：Parquet（方式 B）
  - 选择理由：Parquet 是列式存储格式，压缩效率高，支持类型 Schema，读取速度快，适合金融时间序列数据的存储和分析。

5. 打开 `report.html` 阅读完整报告
